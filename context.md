# Premium Petrol Radar - Context

## Project Objective
The goal of this project is to build an interactive, real-time map and analytics dashboard that helps Indian motorists locate specialized fuel stations. 

Currently, finding high-octane enthusiast fuels (like IndianOil XP100, HPCL Power 99, BPCL Speed 97, and XP95) or standard low-ethanol blends is difficult as there is no unified, free government API or platform that tracks this availability reliably. 

## The Solution
"Premium Petrol Radar - India" solves this by providing a sleek, modern web application where users can:
- View 120+ pre-seeded specialized fuel stations across major Indian cities on a dark-themed interactive map.
- Filter the map dynamically using a 3-position toggle to switch between:
  1. Standard (E20 / E10)
  2. High Octane (XP95 / Speed 97)
  3. Ultra Premium (XP100 / Power 99)
- View rich popups with fuel prices, ethanol blends, and queue statuses.
- Crowdsource new data by dropping pins and reporting fuel availability and pricing.
- View nationwide analytics such as average price deltas between standard and premium fuels.

This tool is designed for the Indian car and bike enthusiast community to track clean fuel.
